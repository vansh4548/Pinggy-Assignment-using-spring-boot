📬 Pinggy Post Manager

A full-stack post management application built using **Next.js (React)** for the frontend and **Spring Boot** for the backend.

---

📁 Project Structure

- **Frontend:** Next.js (React) – Handles the user interface and interaction.
- **Backend:** Spring Boot (Java) – Handles API endpoints and data management.

---
🚀 Getting Started

🧩 Frontend – Next.js

> Ensure Node.js and npm are installed.

1. Navigate to the frontend directory
   cd frontend
2. Install the dependencies
   npm install 
3. Run the Development Server
   npm run dev

Your frontend will be live at http://localhost:3000


☕ Backend – Spring Boot	

Make sure JDK is installed and properly configured. You should also have either Maven or Gradle set up depending on your project build tool.

Run using an IDE (like IntelliJ or Eclipse)
Open the Spring Boot project in your IDE.

Locate the main application class (with @SpringBootApplication annotation).

Right-click the file and select Run.

Your backend server will run at http://localhost:8080
